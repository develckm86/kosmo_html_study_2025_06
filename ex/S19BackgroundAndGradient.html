배경 이미지와 그라디언트

1. background-color

  • 요소의 배경색 지정
  • 값 : 색상 이름, HEX, RGB, HSL 등
  • 예시 
    div {
  background-color: lightblue;
}

2. background-image

  • 요소의 배경에 이미지나 그라디언트 지정
  • 값 : url("이미지경로"), linear-gradient(), radial-gradient() 등
  • 이미지와 그라디언트를 같이 사용시 url을 먼저 작성하고 이미지는 png를 사용해야함
  • 예시 
    div {
            background-image: url("/src/img/apple_logo.png"),
            linear-gradient(#ffffff, lightblue);
}

4. background-size

  • 배경 이미지의 크기 지정
  • 값 : auto, cover, contain, px, %
  • 예시 
    div {
  background-size: cover;
}

3. background-repeat

  • 배경 이미지 반복 여부 지정
  • 값 : repeat, repeat-x, repeat-y, no-repeat
  • 예시 
    div {
  background-repeat: no-repeat;
}

5. background-position

  • 배경 이미지의 위치 지정
  • 값 : left, right, top, bottom, center, px, %
  • 배경은 no-repeat 이면서 사이즈가 고정일때만 위치를 지정할 수 있다.
  • 예시 
    div {
            background-repeat: no-repeat;
            background-size: 5em 5em;
            background-position: 90% 90%;
}

6. background-attachment

  • 배경 이미지의 스크롤 고정 여부 지정
  • 값 : scroll, fixed, local
  • scroll 배경을 요소 자체에 대해 고정합니다
  • local 배경을 요소 콘텐츠에 대해 고정합니다.
  • fixed 배경을 뷰포트에 대해 고정합니다
  • 예시 
    div {
  background-attachment: fixed;
}

7. background-clip

  • 배경이 적용되는 영역 지정
  • 값 : border-box, padding-box, content-box
  • 예시 
    div {
  background-clip: content-box;
}

8. background-origin

  • 배경 이미지의 기준 위치를 지정
  • 값 : border-box, padding-box, content-box
  • 예시 
    div {
  background-origin: padding-box;
}

9. background 단축 속성

  • 여러 background 속성을 한 줄로 작성 가능
  • 순서 : color → image → position/size → repeat → attachment → origin/clip
  • 예시 
    div {
  background: lightblue url("bg.png") no-repeat center/cover fixed;
}
