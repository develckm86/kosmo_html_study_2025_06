트랜지션+트랜스폼 과제

과제 1: 카드 리프트 효과

  • 마우스를 카드에 올리면 살짝 위로 떠오르고 그림자가 깊어진다.
  • transform: translateY(-8px)와 box-shadow를 transition으로 부드럽게 전환.

과제 2: 카드 플립(2D 전용)

  • 앞면·뒷면이 있는 카드를 hover 시 rotateY로 뒤집는다.
  • 각 면은 backface-visibility: hidden 설정.

과제 3: 밑줄 슬라이드 네비게이션

  • hover 시 밑줄이 왼쪽에서 오른쪽으로 확장되는 애니메이션.
  • transform: scaleX(0) → scaleX(1), transform-origin: left 사용.

과제 4: 이미지 줌-인 썸네일

  • 이미지를 hover하면 안쪽으로 확대되며 자연스럽게 보인다.
  • 래퍼에 overflow: hidden, 내부 img는 scale(1.15)로 확대.

과제 5: 툴팁 팝(Scale + Opacity)

  • 아이콘 hover 시 툴팁이 커지면서 나타난다.
  • transform: scale(0.9) + opacity: 0 → scale(1) + opacity: 1로 전환.
