http://localhost:63342/html_study/L26Resource.html

  • http : 통신규약
  • localhost : 내컴퓨터 내부(== 127.0.0.1)
  • 63342 : os 부영한 웹서버의 고유번호 포트(port)
  • html_study/L26Resource.html : 서버내의 리소스 주소 (path)
  • http://localhost:63342/html_study/L26Resource.html : url

상대 경로

  • /path : 루트경로 => 리소스의 맨상단

다른 문서(html)을 포함하는 iframe

/ : 루트 상대경로

./ . ../ : 리소스 위치 기준 상대경로

절대경로 : 통시규약 or 드라이버

절대경로로 다른 사이트 참조

CORP Cross-Origin Resource Policy => SOP : 같은 서버의 리소스만 참조가능
